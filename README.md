# Mitarbeiter-Datenbank
Eine Mitarbeiter Datenbank erstellt im Modul: "Grundlagen der Programmierung" im ersten Semester meines Informatik Studiums

Daten werden in einer separaten Datei gespeichert und wieder ausgelesen.

- Man kann Mitarbeiter hinzufügen/ löschen
- Man kann Mitarbeiter-Daten ändern
- Man kann Mitarbeiter gruppieren
